# Pinned, Starred & Tags

Three orthogonal annotation axes — a clip can carry any combination.

## Why three?

| Axis | Mental model | Where it shows |
|------|--------------|----------------|
| **Pinned** | "Always near the top" | Sticky section atop *All* |
| **Starred** | "I'll come back to this later" | Dedicated *Starred* tab |
| **Tags** | "Group by topic / project" | One tab per tag |

They live on the same `ClipState` row, so toggling one doesn't disturb the others.

## Pinning

- Right-click → *Pin* (or shortcut <kbd>P</kbd>).
- Pinned clips appear in a sticky section at the top of *All*.
- They are exempt from auto-cleanup by age.

## Starring

- Right-click → *Star* (or <kbd>S</kbd>).
- Use it as your "read later" — the **Starred** tab gives a dedicated view.
- Like pins, starred items are kept by auto-cleanup.

## Tags

- Right-click → *Add tag…* — type or pick from existing.
- Clips can have multiple tags.
- Each tag becomes a tab; you can hide / reorder tags from **Settings → Category layout**.
- Bulk-tag via multi-select.
