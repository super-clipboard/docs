# 设置项

> 进入设置：插件右上角 ⚙ 图标，或 uTools 主搜索框搜「Super Clipboard 设置」。
> 所有设置项**首次修改后才会落库**，未改动者保持默认值不写盘。

## 数据库

| 项 | 默认 | 说明 |
|----|------|------|
| 自动过期 | 不过期 | 7 / 14 / 30 / 60 / 120 天可选；超期条目下次启动清理（置顶 / 收藏豁免） |
| 最大保留数量 | 不限制 | 超出数量的最旧条目下次启动清理（同上豁免） |

> 清理在启动后**后台异步执行**，不会拖慢插件冷启动。

## 监听器

| 项 | 默认 | 说明 |
|----|------|------|
| 自动启动 | 开 | 插件加载时自动启动后台剪贴板监听 |
| 监听类型 | 全部 | 可关闭 `image` / `file` 减少噪声 |

后台监听独立于插件窗口，关闭插件窗口不会停止抓取。

## 搜索

| 项 | 默认 | 说明 |
|----|------|------|
| 启用全文搜索 | 关 | 开启后使用内置的高性能索引技术，可在海量历史中秒级检索任意字词；首次开启会后台分批构建索引 |
| 退出后自动清空 | 开 | 插件失焦后延迟清空搜索 / 分类 / 多选 / 预览状态 |
| 清空延迟 | 5 秒 | 0 表示立即清空 |

## 行为

| 项 | 默认 | 说明 |
|----|------|------|
| 复制后隐藏主窗口 | 开 | <kbd>Ctrl/Cmd</kbd> + <kbd>C</kbd> 后自动隐藏 uTools |
| 粘贴前先复制 | 关 | <kbd>Enter</kbd> 粘贴时同时写入系统剪贴板，方便后续手动 <kbd>Cmd/Ctrl</kbd> + <kbd>V</kbd> |

## 偏好

| 项 | 默认 | 说明 |
|----|------|------|
| 外观主题 | 跟随系统 | system / dark / light 三档 |
| 插件高度 | 580 px | 400–1000 px，调整后立即生效（走 `utools.setExpendHeight`） |

## 分类布局

设置 → **分类** 中可调整顶部 Tab：

- 拖拽改变顺序；
- 勾选可见性，隐藏不用的 Tab；
- 标签 Tab 自动出现在末尾，删除标签后 Tab 也消失。

## 日志

| 项 | 默认 | 说明 |
|----|------|------|
| 日志等级 | info | trace / debug / info / warn / error |

日志保留 15 天，存储在系统的日志目录下。

## 脚本

设置 → **脚本** 进入脚本管理页：

- **已安装** 列表：启停 / 卸载 / 查看授权 / 触发更新；
- **市场**：浏览官方仓库，一键安装；
- **新建**：内置轻量级代码编辑器（仅提供语法高亮，如需复杂编辑请使用外部编辑器后复制回来），保存即生效。

详见 [脚本系统概述](/zh/scripts/overview)。
