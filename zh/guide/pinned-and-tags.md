# 置顶、收藏与标签

三套独立的高亮维度，覆盖「常用」、「重要」、「分类」三种诉求。

## 置顶（Pinned）

- **目的**：让常用条目恒在分类顶部、不被新内容挤出视野。
- **入口**：右键菜单 → *置顶*；或选中后 <kbd>Ctrl/Cmd</kbd> + <kbd>P</kbd>。
- **效果**：所有 Tab（除「收藏」）都会把置顶项排在最前；图钉徽标常驻。
- **不影响清理**：自动过期 / 最大数量上限会**跳过**置顶项。

> 多选下按 <kbd>Ctrl/Cmd</kbd> + <kbd>P</kbd> 会批量切换：若全部已置顶则全部取消，否则全部置顶。

## 收藏（Starred）

- **目的**：标记长期保留的内容，并附加一段**备注**。
- **入口**：右键菜单 → *收藏*。
- **「收藏」分类 Tab**：仅显示带星标的条目。
- **备注字段**：仅收藏的条目支持，按 <kbd>Ctrl/Cmd</kbd> + <kbd>E</kbd> 进入编辑面板写入。
  取消收藏时备注**会一并清除**（这是有意为之的状态压缩）。

> 「置顶」与「收藏」相互独立：可以只置顶不收藏，或反之。

## 标签（Tags）

- **目的**：自定义彩色分组，例如「邮件签名」「常用脚本片段」「项目 A 链接」。
- **创建**：在编辑面板里输入新标签名，按 <kbd>Enter</kbd> 创建并自动分配颜色。
- **复用**：编辑面板列出已有标签，点击即可加入或移除。
- **管理**：设置 → 标签 中可重命名 / 删除标签；删除后所有关联条目失去该标签，**不可恢复**。

每个标签自动成为一个分类 Tab，按 <kbd>Tab</kbd> 切换即可只看该分组。
