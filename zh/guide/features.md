# 功能总览

一页内列出 Super Clipboard 提供的全部功能；点击任一项跳转细节页。

## 历史记录

- **三类内容统一管理**：文本 / 图片 / 文件 三种 clip 类型独立 Tab。
- **内容哈希去重**：同一段内容反复复制只占一条记录，但「最近使用时间」会刷新到顶。
- **左侧时间列**：每条记录在左侧显示相对时间（刚刚 / 5 分钟前 / 昨天…）。
- **顶部分类 Tab**：可在 [设置 → 分类布局](./settings#分类布局) 中拖拽顺序、隐藏不用的 Tab。
- **底部加载更多**：滚动到底自动追加，无需翻页。

→ 详见 [历史列表](./history-list)

## 检索

- **顶部 utools subInput**：在历史页直接键入即聚焦搜索框（中文输入法兼容）。
- **全文搜索**：可在设置中开启 SQLite FTS 索引（首次启用会后台分批构建）。
- **退出后自动清空**：默认 5 秒后清掉搜索框、分类、多选、预览状态，下次进入是干净状态。

## 快速操作

- **<kbd>Enter</kbd> 粘贴**：选中条目后回车，自动写入剪贴板并触发系统粘贴。
- **<kbd>Ctrl/Cmd</kbd> + <kbd>C</kbd> 复制**：仅写入剪贴板，不触发粘贴。
- **<kbd>Space</kbd> 进入多选**：连续按空格逐项勾选；多选下 <kbd>Enter</kbd> 合并粘贴。
- **<kbd>Ctrl/Cmd</kbd> + <kbd>P</kbd> 置顶**、**<kbd>Ctrl/Cmd</kbd> + <kbd>E</kbd> 编辑**。



## 置顶 / 收藏 / 标签

- **置顶**：任何条目可置顶，置顶项浮在分类顶端。
- **收藏**：被收藏的条目支持额外的「备注」字段；取消收藏会自动清除备注。
- **标签**：自定义彩色标签，单个条目可多标签；标签出现在分类 Tab 中。

→ 详见 [置顶、收藏与标签](./pinned-and-tags)

## 预览面板

- **段落点选复制**：长文本会按段落渲染，按住 <kbd>Cmd</kbd> / <kbd>Ctrl</kbd> 点击任一段落即复制该段。
- **图片预览**：渲染原图（缩略图直接来自 meta 的 base64 preview，原图按需加载）。
- **文件列表**：列出 clip 内全部文件路径与大小。

→ 详见 [预览与段落点选复制](./preview-and-paragraph-copy)

## 链接快速打开

- **悬停高亮**：文本中识别 URL（白名单：`http`/`https`/`ftp`/`mailto`/`tel`/`ws`/`wss`）后高亮。
- **<kbd>Cmd/Ctrl</kbd> + 点击**：在系统默认浏览器中打开。
- **右键菜单**：「在浏览器打开」「复制链接」。

→ 详见 [链接快速打开](./link-quick-open)

## 脚本扩展

- **TypeScript-first**：附带 `@super-clipboard/userscript-types` 类型包，IDE 提示与跳转开箱即用。
- **沙箱执行**：每个脚本运行在独立 iframe（`sandbox="allow-scripts allow-downloads"`）。
- **两种粒度的 Grant**：`utools.*` 暴露 uTools 子集；`globalNativeApi.*` 暴露剪贴板 / KV / 面板 / 文件 API。
- **内置脚本**：二维码、智慧分词、另存为、自动 OCR 标注…… 见 [示例集](/zh/scripts/examples)。

→ 详见 [脚本系统概述](/zh/scripts/overview)

## 数据治理

- **自动过期**：可设 7/14/30/60/120 天，过期项启动时清理（置顶 / 收藏不计入）。
- **最大数量上限**：超过上限的最旧条目启动时清理（同样豁免置顶 / 收藏）。
- **存储与同步**：所有数据存于 uTools 本地 DB；如果开通了 uTools 会员，
  则可以随 uTools 会员服务在多设备间同步历史。

## OCR 图片识别

- **macOS**：内置使用系统 Vision Framework，离线运行，无需额外设置。
- **其他平台**：可通过安装 [脚本市场](/zh/scripts/overview) 中的 OCR 脚本接入云服务实现，
  常见选项包括百度、讯飞、腾讯云、阿里云 OCR 等。

OCR 结果会写回到对应图片 clip，后续可以被搜索到。
