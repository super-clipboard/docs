# 预览与段落点选复制

按 <kbd>`</kbd>（反引号）切换右侧 **预览面板**。预览面板针对三种 clip 类型有不同呈现。

## 文本预览

- 默认按段落渲染（保留换行 / 列表 / 代码块），并对超长段落分段折叠。
- **段落点选复制**：
  - **macOS**：按住 <kbd>⌘</kbd> 点击任一段落 → 仅复制该段落。
  - **Windows / Linux**：按住 <kbd>Ctrl</kbd> 点击。
  - 鼠标悬停在段落上时，按住修饰键会有视觉高亮提示。
- 选中文本后按 <kbd>Ctrl/Cmd</kbd> + <kbd>C</kbd> 走系统原生复制；不会触发列表层的「复制整条」。

## 图片预览

- 列表里展示的缩略图直接来自 `ClipMeta.preview` 字段（≤ 200px 边长的 base64），首屏零 IO。
- 进入预览面板后才按需读取原图字节（`getAttachment('clip-body/<hash>')`），
  保证大图不阻塞主列表滚动。
- 图片预览支持旋转 / 翻转 / 缩放（工具栏按钮）；右键 *另存为* 下载到本地。

## 文件预览

- 列表里只展示前若干个文件名；预览面板列出全部 `path` / `name` / `size` / `mtime`。
- 单击任一文件路径即复制该路径；按住 <kbd>Cmd/Ctrl</kbd> 点击在 Finder / 资源管理器中显示。

## 链接识别

预览面板里的文本会自动识别 URL 并渲染为可点击链接，
详情见 [链接快速打开](./link-quick-open)。
